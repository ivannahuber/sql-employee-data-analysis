# SQL Employee Data Analysis

## Project Overview

This project demonstrates SQL fundamentals using Microsoft SQL Server.

The project analyses employee salary and department data by answering common business questions using SQL queries.

## Tools Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- SQL

## Skills Demonstrated

- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- LIKE
- BETWEEN
- IN
- GROUP BY
- Aggregate Functions (SUM, AVG, MIN, MAX, COUNT)
- Subqueries
- CASE Statements
- Common Table Expressions (CTEs)
- Window Functions (RANK)

## Dataset

The dataset contains 15 employees with the following information:

- Employee ID
- First Name
- Last Name
- Department
- Salary

## Business Questions Answered

This project answers questions such as:

- Who are the highest-paid employees?
- Which department has the highest average salary?
- How many employees work in each department?
- Which employees earn above the company average?
- What is the total salary paid by each department?

- `employee_data_analysis.sql` – Complete SQL project including table creation, sample data and analysis queries.
